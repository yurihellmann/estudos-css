Seletores CSS

Bem-vindo

  • Seletor por Tipo: busca elementos por uma tag HTML especifica. Ex.: h1, h2, ul, li etc.

  • Seletor por ID (#): busca elementos através do atributo "id".

  • Seletor por Classe (.): busca elementos através do atributo "class".

  • Seletor Universal (*): seleciona TODOS os elementos do CSS.

  • Seletor de Atributo ([atrib]): seleciona elemtnos que possuem um atributo específico no HTML. É possível buscar atributos por um valor específico.
