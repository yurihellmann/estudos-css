Agrupameno de Seletores

É possível aplicar as regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula.

Assim, não é encessário copiar e colar os mesmos códigos para todos os elementos, pode ser formado um "bloco" só de CSS e reutilizar as mesmas regras.
